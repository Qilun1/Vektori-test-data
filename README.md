# Vektori

**E-commerce Analytics & Experimentation Platform**

Vektori helps online stores understand customer behavior, optimize conversion funnels, and run A/B tests. We serve ~120 active customers across Europe, mostly mid-market e-commerce brands.

## What We Do

- **Real-time dashboards** -- Live event tracking and conversion metrics for e-commerce stores
- **Funnel analysis** -- Visual conversion funnel builder with drop-off analysis
- **A/B testing** -- Full experimentation platform with statistical significance testing
- **Cohort reports** -- Behavioral and time-based retention analysis
- **Predictive analytics** -- ML-based churn prediction (shipped Q1 2026)

## Integrations

Vektori integrates with major e-commerce platforms:

- Shopify
- WooCommerce
- Magento
- Stripe

Data can be collected via our [JavaScript SDK](sdk/README.md) or the [REST API](docs/api-reference.md).
